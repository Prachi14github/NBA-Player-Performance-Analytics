# NBA Player Performance Analytics Dashboard

## About the Project

This project focuses on analysing player and team performance using the 2024/25 NBA dataset. The dashboards were created in Tableau and Power BI to explore scoring trends, compare player statistics, and present the data through interactive visualisations.

## Tools Used

* Tableau
* Power BI
* DAX
* Microsoft Excel

## Dataset

* NBA 2024/25 Season Dataset

## Features

* Player and team performance analysis
* Interactive filters and dashboards
* Scoring trend analysis
* Performance comparison
* KPI visualisations
